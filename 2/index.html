    Cara Buat Folder Pada Github
  1. Buka website github
  2. Klik new repositori
  3. Buat nama repositori(si5a2122o)
  4. Isi deskripsi
  5. Pilih Public
  6. Pilih add a README file
  7. Klik Create Repositori
  8. Muncul lah File README
  9. Pada add file klik create new file
  10. Ketik angka 1 / (untuk nama folder)
  11. Lalu ketik index.html (nama file)
  12. Masukkan isi
  13. Klik Commite new file
  14. Jika ingin tambah foto dan ubah foto jadi link, contoh ketik script seperti pada gambar berikut
  15. Klik add files, upload files, lalu drag foto dan klik commit
  16. Jika sudah, klik settings
  17. Klik Pages
  18. Source none diganti main
  19. Klik save
  20. Muncul link, lalu klik
  21. Buka link tersebut dengan menambahkan nama folder yang kita buat tadi (delaper.github.io/si5a21220/1/)

    Penjelasan admin area & client area
  1. Wordpress terdiri dari 2 sisi, yaitu site preview/client area (halaman depan/frontend, hasil dari admin area) dan admin area (backend/dasbor admin)

    URL Login
  1. Admin bisa login di link, http://localhost/si5a2122o/wp-admin

    Cara Ubah Profil & Password
  1. Login pada halaman admin
  2. Selanjutnya ubah profil password
  3. Pada dasbor, klik menu pengguna, klik menu profil
  4. Penyunting visual(mode penyuntingan pakai klik-klik tanpa codingan)
  5. Jika ingin upload foto, klik tanda tambah, ketik gambar, lalu klik ikon gambar. Kita bisa mengunggah dari sendiri dan bisa juga dari URL. Contoh seperti gambar dibawah
  6. Penyorot syntax(berhubungan dengan penyunting visual)
  7. Skema warna bisa ditentukan sesuai selera
  8. Pintasan keyboard(Jika web kita sudah banyak komentar dari pengunjung, akan terbantu jika menggunakan pintasan keyboard seperti shortcut control)
  9. Bilah perkakas(Untuk mempermudah edit/sunting web). Namun jika laptop kita digunakan oleh banyak orang disarankan untuk menonaktifkan bilah perkakas.
  10. Nama pengguna diganti dengan nama masing-masing
  11. Membutuhkan main server yang aktif untuk mengirim sebuah email
  12. Isi info biografi
  13. Untuk foto profil, bisa daftar dulu di wordpress.com
  14. Lalu atur kata sandi
  15. Jika sudah, klik perbarui profil

    Daftar User Baru
  1. Klik menu pengguna, klik menu tambah baru
  2. Masukkan data pengguna baru
  3. Pada peranan, pelanggan(pelanggan bisa melihat rincian aktivitas di dasbor, kemampuannya hanya bisa update profil & password), kontributor(kontributor bisa melihat komentar,
    bisa melihat semua post tapi tidak bisa mengganggu pos tersebut, bisa menambah post baru namun tidak langsung bisa muncul di halaman depan tapi harus melalui persutujuan minimal dari editor & admin dulu,
    setelah disetujui admin postnya tidak bisa diedit lagi oleh kontributor karena sudah diambil alih oleh admin, kontributor harus mengambil media dari tempat lain), Author(bisa buat pos tanpa tinjauan dari admin dan bisa langsung tampil di web,
    media bisa langsung diupload dari web server), Editor(Bisa menyunting postingan, orang kepercayaan admin, tidak bisa ganti tampilan), Administrator(Punya hak akses tertinggi, bisa instal plugin, bisa ganti tampilan

    Pengaturan/Setting
  1. Pada menu umum, Judul situs dan slogan, ditentukan sendiri
  2. Keanggotaan(ada banyak pengguna dalam satu akun untuk memposting web. Akan ada link daftar pada halaman login)
  3. Zona waktu ubah ke UTC+7
  4. Format waktu bisa disesuaikan
  5. Jika sudah, klik simpan
  6. Pada menu menulis, kategori post standar bisa ditambahkan
  7. Bisa posting via email tetapi perlu dipertimbangkan dulu karena perlu melampirkan password
  8. Pada menu membaca, berhubungan dengn pengunjung web.
  9. Jika ingin menampilkan post terbaru maka klik pos-pos terbaru Anda pada tampilan beranda anda
  10. Untuk setiap pos dalam feed biasanya menggunakan kutipan
  11. Pada menu diskusi, membahas tentang pengaturan komentar
  12. Pada menu media, file foto yang diupload di dalam folder upload kita terdapat berbagai macam ukuran foto yang diambil oleh wordpress untuk pengoptimalan web. Semakin besar ukuran gambar
    maka resolusinya semakin besar dan waktu loadingnya juga lama begitupun sebaliknya.
  13. Pada menu permalink, pengaturan tentang nama link sesuai dengan yang dibutuhkan
  14. Pada menu privasi, semacam disclaimer.Pengaturan tentang cookie.
