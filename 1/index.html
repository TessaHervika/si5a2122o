Pada pertemuan 1 belajar cara install, backup, dan restore Wordpress.
    Cara Install Wordpress
  1. Download file zip wordpress di halaman wordpress.org
  2. Ekstrak file zip tersebut
  3. Rename nama folder (usahakan nama folder jangan ada spasi, contoh: si5a2122o)
  4. Xampp control harus dalam keadaan learning
  5. Pindah file ke folder xampp -> htdocs
  6. Ingat nama folder saat akan di akses. Contoh (localhost/si5a2122o)
  7. Setelah di akses pilih bahasa
  8. Siapkan database dengan membuka xampp control lalu klik admin pada MySQL
  9. Akan muncul halaman localhost/phpmyadmin
  10. Isi keterangan koneksi basis data (pastikan databasenya sudah ada di phpMyAdmin
  11. Klik kirim & Jalankan Sekarang
  12. Masukkan Informasi yang dibutuhkan
  13. Klik Instal Wordpress
  14. Setelah berhasil coba login
  15. Web pun sudah jadi

    Cara Backup Wordpress
  1. Buat satu buah folder baru
  2. Copy folder yangg sudah di rename saat instal tadi (contoh: si5a2122o)
  3. Paste di folder yang baru
  4. Lalu dari phpMyAdmin klik export, pilih custom dan pilih database yang telah kita buat tadi
  5. Klik tombol Go
  6. Simpan di folder baru tadi
  7. Backup selesai
  8. Compress file-file tersebut menjadi satu folder

    Cara Restore Wordpress
  1. Ekstrak folder yang telah di backup
  2. Lalu import backup database
  3. Pilih file database
  4. Klik Go
  5. Jika sudah berwarna hijau, restore pun berhasil
