    Posting (Konten Web)
  1. Klik Pos, tambah baru. Lalu tambahkan judul.
  2. Isi bagian teks sesuai keinginan kita.
  3. Lalu edit tulisan sesuai keinginan dengan editor blok secara visual maupun html.
    Jika ingin mengubah ke mode html, klik titik tiga lalu klik sunting sebagai html.
  4. Jika ingin menambah tabel, searching tabel lalu masukkan jumlah kolom dan baris klik buat tabel.
    Jika ingin mengubah header tabel maka ubahlah ke penyuntingan html dan tambahkan scriptnya
  5. Untuk menyempurnakan block maka digunakanlah HTML.
  6. Klik terbitkan untuk menambah postingan dan muncul di frontend
  7. Jika ingin mengubah klik semua pos, cari judul lalu klik sunting dan edit sesuai keinginan.
  8. Jika ingin menghapus postingan klik tong sampah

    Cara buat draft (Konsep)
  1. Tambah postingan, lalu ketik judul dan isi.
  2. Klik simpan konsep
  3. Bisa dilanjutkan kapanpun tulisannya.
    4. Status Publish ( sudah diterbitkan dan sudah muncul)
    Pending review (tulisan oleh kontributor yang tidak langsung muncul dan perlu ditinjau oleh editor & admin)

    Cara buat Scheduled
  1. Tambah pos baru
  2. Ketik judul dan isinya
  3. Lalu klik Pos pada kanan layar, klik terbitkan->Secepatnya
  4. Tentukan sesuai tanggal yg kita inginkan
  5. Klik jadwal lalu penjadwalan

    6. Untuk mengizinkan siapa saja yg bisa melihat blog, bisa pilih di visibilitas.

    Cara buat kategori
  1. Buka postingan
  2. Klik pengaturan pada pos
  3. Klik tambah kategori baru
  4. Satu post bisa dibuat dalam beberapa kategori

    Cara buat Menu
  1. Klik tampilan->menu
  2. Pilih menu yg akan diedit, klik pilih
  3. Klik kelola lokasi
  4. Pada menu horizontal desktop pilih menu yg kita pilih tadi
  5. Klik simpan perubahan
  6. Jika ingin menambahkan kategori pada menu maka, klik kategori pada pengaturan menu
  7. Pilih kategori dan klik tambahkan ke menu. Kategori akan muncul
  8. Jika ingin membuat sub item tinggal tarik ke kanan misalnya kontak sub dari blog.
  9. Klik simpan
  10. Jika ingin menambahkan link lain(link eksternal) pada menu kita, salin tautan linknya dulu.
  11. Masuk ke tampilan->menu, lalu klik Taut Tersuai. Masukkan Linknya dan beri nama teks
  12. Klik Tambahkan ke menu dan simpan menu

    Cara buat tags/label
  1. Sunting postingan
  2. Klik tag pada pos, dan masukkan tag yg kita inginkan
  3. Klik perbarui

    Cara memunculkan/mengaktifkan komentar
  1. Sunting postingan
  2. Klik diskusi lalu centang izinkan komentar pada pos
  3. Klik perbarui

    Cara Unggah media
  1. Klik media, tambah media baru
  2. Drag file yg ingin kita tambahkan dari "everything Voidtools"
  3. Buat postingan baru dan masukkan file tersebut melalui berkas-> pustaka media.
